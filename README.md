## 使い方
- ZIPをダウンロードする場合  
Clone or Download→Download ZIP→ZIPを展開  
- Gitをインストール済みの場合  
```
git clone https://github.com/pb10001/utakata-umigame.git
```
- ライブラリをインストールする
```
npm install
```
- ローカルサーバーをスタートする
```
node server.js
#localhost:5000にアクセス
```
